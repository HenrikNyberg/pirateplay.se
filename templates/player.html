Pirateplayer

Pirateplayer är ett fritt, grafiskt program avsett att spela upp och ladda ner svensk webb-TV i Windows, Mac OSX och Linux.

Pirateplayer kan ladda ner ifrån: ${s.title},

Alltså; samma tjänster som Pirateplay.se stöder. Finns undertexter försöker Pirateplayer i många fall även ladda ner dem.

Pirateplayer använder biblioteken Qt, librtmp, programmet Ffmpeg och API:n från Pirateplay.se.

Ladda ner och installera

Windows

Ladda ner och kör installationsprogrammet. Pirateplayer finns därefter tillgängligt från Startmenyn

Archlinux

0.4.2 är på väg.

Installera från med hjälp av PKGBUILD via AUR.

Debian/Ubuntu

0.4.2 är på väg.

  • Kör sudo aptitude install libqtcore4 libqtgui4 libqt4-xml libqt4-network phonon libphonon4 rtmpdump libqt4-svg libqtwebkit4
  • Hämta deb-paktet.
  • Gå till katalogen du hämtade deb-paktet till och kör sudo dpkg -i pirateplayer_0.3.1-1_i386.deb
  • Pirateplayer är nu installerat och kan köras med pirateplayer

Obs: vill du kunna se/ladda ner från TV4-play måste Rtmpdump på din dator vara av version 2.1d eller nyare än version 2.3.

Mac OSX

0.4.2 är på väg.

Installera Apple diskavbild (dmg).

Tyvärr fungerar inte videouppspelning på Mac i nuläget. Åtgärdas förhoppningsvis framöver.

Källkod

Finns inte ditt operativsystem/distribution ovan kan du alltid kompilera och installera från källkoden.

Hör gärna av dig om du har möjlighet att bygga/testa paket för din favoritdistribution (chucky snabela wrutschkow.org).

Historik

Version 0.4.2 (2012-09-18)

  • Buggfix: stökiga sökvägar i dialogrutan för att spara strömmar.
  • Buggfix: mer konsekvent visning av sökvägar i Windows.
  • Buggfix: "Öppna" nedladdning i Windows.
  • Suffixförslag: Pirateplayer försöker föreslå lämplig filändelse.

Version 0.4.0 (2012-09-17)

  • Stöd för HLS-strömmar (många av SVTs videos är numera av den typen).
  • Reorganisation av användargränssnittet.
  • Redigering av nedladdningsaddresser.
  • Uppspelning av strömmar i extern videospelare.
  • Videospelaren borttagen: många externa videospelare har numera möjlighet att spela upp RTMP/HLS-strömmar.
  • Visning av nyheter (via Twitter).
  • Möjlighet att återuppta avbrutna nedladdningar borttagen.
  • Användarinställningar.

Version 0.3.1 (2012-01-14)

  • Applikationsikon - tack till Bernard!

Version 0.3 (2012-01-11)

  • Viss möjlighet att återuppta avbrutna nedladdningar.
  • Inbyggd programväljare, ett slags fjärrkontroll.
  • Hantering av ompekande http-svar - gör att nedladdningar från Sveriges Radio fungerar.
  • TV4s undertexter sparas med filändelsen smi (tidigare smi?) för att inte fucka ur på Windows-system.
  • Buggfix: Nedladdning av program på över en timme avbryts ej längre.

Version 0.2 (2011-11-11) 

  • Parallella nedladdningar.
  • Inbyggd videospelare.
  • Nedladdning av undertexter.
  • Flertalet buggfixar.
  • Installationsprogram för Windows.

Version 0.1 (2011-10-12)

  • Första versionen!

Kända buggar

Ladda ner från SR

Fungerar inte i nuläget. Kommer åtgärdas i nästkommande version. Fungerar sedan version 0.3!

Videouppspelning Mac

Fungerar inte i nuläget. Funktion borttagen sedan version 0.4.0.

32-bitars Mac datorer

Binären borde vara byggd som "universell". Pirateplayer fungerar i nuläget bara på 64-bitars macar.
