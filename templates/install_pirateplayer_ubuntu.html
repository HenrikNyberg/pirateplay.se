Skaffa rätt debianpaket

Nedan följer en lista över färdigbyggda debianpaket för Pirateplayer 0.4.6 och libtidy-gitversion. Ladda ner de som hör till din version av Ubuntu/Debian. Finns inte din version kan du testa med den version som ligger närmast. Fungerar det ändå inte kan du kompilera Pirateplayer själv.

  • Ubuntu 12.10 32-bitars:
    • pirateplayer_0.4.6-1_i386.deb
    • libtidy_20130210-git-1_i386.deb
  • Ubuntu 12.04 64-bitars:
    • pirateplayer_0.4.6-1_amd64.deb
    • libtidy_20130210-git-1_amd64.deb
  • Ubuntu 12.04 32-bitars:
    • pirateplayer_0.4.6-1_i386.deb
    • libtidy_20130210-git-1_i386.deb

Stort tack till Milleman för Ubuntu 12.04 paketen!

Installera beroenden

  1. sudo add-apt-repository ppa:jon-severinsson/ffmpeg
  2. sudo apt-get update
  3. sudo apt-get upgrade -y
  4. sudo apt-get install libqtcore4 libqtgui4 \ libqt4-xml libqt4-network libqt4-declarative libqt4-script \ libqt4-xmlpatterns libqt4-svg libqtwebkit4 rtmpdump librtmp0 \ ffmpeg xsltproc -y

Installera Libtidy och Pirateplayer

Ersätt [version] och [arkitektur] i kommandona nedan så de stämmer överens med de filnamnen för de debianpaket du laddade ner tidigare.

  1. sudo apt-get remove libtidy*
  2. sudo dpkg -i libtidy_[version]-git-1_[arkitektur].deb
  3. sudo dpkg -i pirateplayer_0.4.6-1_[arkitektur].deb

En sista touch

För att Pirateplayers tillägg ska kunna använda sig av systemets egna kontroller (knappar, textrutor o.s.v.) måste Qt Desktop Components vara installerat. Detta är dock inget krav för att tilläggen ska fungera. Nedanstående anvisningar för att installera Qt Desktop Components fungerar i Ubuntu 12.04 - får du det inte att fungera i ditt system kan du hoppa över detta steg - i Ubuntu 12.10 fungerar det t.ex. inte. Pirateplayer kommer fungera ändå.

  1. sudo add-apt-repository ppa:forumnokia/fn-ppa
  2. sudo apt-get update
  3. sudo apt-get install qt-components-desktop -y
